# SortingDirection<a name="enum-sortingdirection"></a>

Enumeration defining sorting direction.
## Enumeration Literals

### ASC - (0)

<em>Sorts the data from the lowest value to the highest value (e.g., A to Z, 1 to 10, oldest date to newest date). This is the default sort order if no keyword is specified.</em>

### DESC - (1)

<em>Sorts the data from the highest value to the lowest value (e.g., Z to A, 10 to 1, newest date to oldest date).</em>

### NONE - (1)

<em>Sorts the data as they come from database</em>
